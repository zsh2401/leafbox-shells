The Leaf Box is an AI application that based on web technologies.
All people can access it with modern browser at https://ai.zsh2401.top.

Nonetheless, for better distribution, we decide to develop shells for Leaf Box.

Leaf Box will be wrapped by native webview components to erase difference between 
browsers. 

Almost all platform is targeted for the development of shells of Leaf Box. 
Android, Windows, Linux and macOS are all supported at first time.
